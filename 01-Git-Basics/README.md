## What is Git?

Git is a distributed version control system that tracks changes in files over time.

It helps developers:

- Track changes
- Maintain project history
- Collaborate with others
- Restore previous versions
- Create multiple branches

Instead of saving multiple folders like:

- Project_Final
- Project_Final2
- Project_Final_Last
- Project_Final_Last_Final

Git stores every change efficiently inside a repository.
