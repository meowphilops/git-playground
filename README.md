# Git Playground :rocket:

> Learn git by actually breaking things, fixing them and understanding that what Git is doing behind scenes.

---

## About

This repository is my personal Git learning playground.

Instead of just memorizing Git commands, this repository focuses on understanding:

- Why a command exists
- When to use it
- What happens internally
- Common mistakes
- Real-world developer workflows

Every folder contains a separate lab where I intentionally create scenarios, make mistakes, and learn how to recover from them.

This repository will continue growing as I learn more advanced Git concepts.
